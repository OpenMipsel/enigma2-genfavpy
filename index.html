Enigma2-genfavpy

Download as .zip Download as .tar.gz View on GitHub

DESCRIÇÃO

Este aplicativo gera as configurações de favoritos do Enigma2.

Ele lê a base de dados de canais do enigma2 (/etc/enigma2/lamedb) e gera as configurações de favoritos com base em um conjunto de regras que podem ser facilmente personalizadas.

Ele funciona tanto para quem aponta apenas para um satelite ou para vários satelites, de acordo com a configuração presente no aparelho (satellites.xml).

OBS: Este aplicativo veio para substituir o aplicativo genfav.sh

COMPATIBILIDADE

Este script foi testado no OpenPLI mas deve funcionar nas demais distribuições.

MODO DE USO

1) FAÇA BACKUP DA PASTA "/etc/enigma2"

2) Copie o arquivo "genfavpy.py" para a pasta "/home/root" no enigma2 usando um cliente FTP (Ex: Filezilla)

3) Instale os pacotes Python necessários:
opkg install sqlite3 python-sqlite3

Caso não consigam instalar pelo comando acima, baixem os pacotes do link abaixo:

https://www.dropbox.com/sh/y6ks6mtu88340mf/jcQCrstqHc

Agora é só instalar manualmente nesta ordem:
opkg install libsqlite3-0_3.6.23.1-r4.0_mipsel.ipk
opkg install sqlite3_3.6.23.1-r4.0_mipsel.ipk
opkg install python-sqlite3_2.6.6-ml12.6_mipsel.ipk

4) Dê permissão de execução a este arquivo. Acesse via Telnet ou SSH o enigma2 e execute o seguinte comando:
chmod 700 /home/root/genfavpy.py

5) Descompacte (WinRAR) o arquivo "genfavpy.conf.tar.gz" e copie o arquivo "genfavpy.conf" para a pasta "/etc/enigma2" no enigma2 usando um cliente FTP (Ex: Filezilla).

6) Na interface do enigma2 (via controle remoto), execute um scan completo de canais para cada satelite que se tenha apontamento.

7) Execute o aplicativo para que ele crie os favoritos baseados nas regras contidas no arquivo "genfavpy.conf". Acesse via Telnet ou SSH o enigma2 e execute o seguinte comando:
/home/root/genfavpy.py

Feito os procedimento acima, em caso de mudança nas TPs, faça:

1) FAÇA BACKUP DA PASTA "/etc/enigma2"

2) Efetue, se necessário, as alterações nas configurações do satellite.xml e reinicie o aparelho (vide forum).

3) Na interface do enigma2 (via controle remoto), execute um scan completo de canais para cada satelite que se tenha apontamento.

4) Execute o aplicativo para que ele crie os favoritos baseados nas regras contidas no arquivo "genfavpy.conf". Acesse via Telnet ou SSH o enigma e execute o seguinte comando:
/home/root/genfavpy.py
